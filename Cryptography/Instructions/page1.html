Char


We haven't talked about the "char" primitive type much, but it holds a single character in it, such as A or c or 5(but not multiple characters, the way a String does).

Chars are interesting. Create a main method somewhere and try the following:

char thisChar = 'C';
System.out.println(thisChar);

Note the single quotes around the 'C', which indicate a single character instead of a String.
